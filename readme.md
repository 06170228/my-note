## 心得


這學期的課程我想就是這門課最令我頭疼吧！因為以前的python基礎沒有打穩，導致這學期要花更多時間要還這些技術債，如同老師所說的逃避不是解決辦法的方式。在這邊特別感謝周柏佳同學和厲彥伯同學，兩位同學非常有耐心的教導我、與我一同討論。周同學知道我對於不會的東西，一直很逃避很消極，她鼓勵我並說要跟我一起努力，讓我想到老師在Facebook上打的：一起打過程式的朋友，才是真朋友 இдஇ


從第一次作業，根本不知道要從何開始打起，因為以前python完全沒學好（其實以前蠻混的><)到最後一次有漸漸進入狀況。其中有許多同學對老師及助教反應上課方式的問題，其實我發現老師真的有在改變上課模式，真的有聽見學生的反應，像是請同學上台實作，了解學生的問題點在哪裡，看到老師的改變很感動！但我想建議一下，因為兩班的助教不同，A班助教講的事情，B班同學不會知道，雖然之後有在ppt上公告一些事項，但還是希望助教能是同一個，兩班才能比較統一（我知道助教真的不好當，薪水也不多，還要改兩班的作業，真是辛苦司助教了～）


最後總結一句：這是一門會爆肝的課！
